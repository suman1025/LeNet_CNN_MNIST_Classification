# LeNet_CNN_MNIST_Classification
## Project Overview
This project was completed to understand how the LeNet CNN model works in handwritten digit image classification. The MNIST dataset was used in this project to train and test the CNN model using TensorFlow and Keras. The project includes image preprocessing, model creation, model training, and accuracy evaluation.
By completing this project, I learned how CNN models recognize image patterns and classify handwritten digit images from 0 to 9. This project also helped me improve my understanding of deep learning implementation using Jupyter Notebook.

## Project Objectives
The main objectives of this project are:
- To understand the basic structure of the LeNet CNN model.
- To learn how CNN models process image data.
- To use the MNIST dataset for handwritten digit classification.
- To train the CNN model using TensorFlow and Keras.
- To evaluate model performance using accuracy results.
- To understand image preprocessing and model training steps.

## Dataset Information
The MNIST dataset was used in this project. The dataset contains handwritten digit images from 0 to 9. Each image size is 28 × 28 pixels. The dataset includes both training and testing images for image classification tasks.

## Tools and Technologies Used
The following tools and technologies were used in this project:
- Python
- TensorFlow
- Keras
- Matplotlib
- Jupyter Notebook

## Data Preprocessing
The MNIST dataset was loaded into training and testing datasets. The image pixel values were normalized to improve model performance. The images were reshaped because CNN models require image data in four-dimensional format.

## Exploratory Analysis
Some sample MNIST digit images were displayed to better understand the dataset. This helped visualize the handwritten digit patterns used for model training.

## Model Implementation
The LeNet CNN model was created using convolution layers, pooling layers, flatten layer, and dense layers. The convolution layers helped the model learn image features, while the pooling layers reduced image size. The dense layers were used to classify handwritten digit images.

## Model Training and Evaluation
The LeNet CNN model was trained for 20 epochs using the MNIST training dataset. During training, the model learned image patterns and improved prediction accuracy. The final accuracy result showed that the model correctly classified most handwritten digit images.

## Key Findings and Analytical Perspective
This project showed that CNN models can perform very well in image classification tasks. The LeNet CNN model achieved high accuracy on the MNIST dataset after training for 20 epochs. The project also showed the importance of preprocessing, convolution layers, and model evaluation in deep learning projects.

## Conclusion
This project helped me understand how CNN models are implemented for image classification tasks. The LeNet CNN model successfully classified handwritten digit images from the MNIST dataset with high accuracy. This project also improved my understanding of TensorFlow, CNN architecture, and deep learning workflows.
